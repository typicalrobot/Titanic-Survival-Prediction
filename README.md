# Titanic Survival Prediction

## Overview
This project predicts whether a Titanic passenger survived using machine learning models in Python.

## Tools Used
- Python
- pandas
- scikit-learn
- matplotlib
- Visual Studio Code

## Dataset
Seaborn Titanic dataset

## Models Used
- Logistic Regression
- Decision Tree Classifier

## Process
1. Loaded the Titanic dataset
2. Selected useful features
3. Filled missing values
4. Encoded categorical variables
5. Split the data into training and testing sets
6. Trained two machine learning models
7. Compared model performance using accuracy, classification report, and confusion matrix
8. Visualized feature importance and survival trends

## Results
- Logistic Regression performed well for predicting survival
- Decision Tree provided a second model for comparison
- Sex and passenger class were the most important features
- Women and first-class passengers were more likely to survive
